# Forsees Consulting

Marketing site for Forsees Consulting, an independent consultancy that
focuses on private/public contracting, IT infrastructure, Real Estate Consulting, and educational strategy,


## Tech stack

- [TanStack Start](https://tanstack.com/start) (React 19 + TanStack Router)
- Tailwind CSS 4
- Vite 7
- Deployed on Netlify

## Running locally

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:3000`.

To build for production:

```bash
npm run build
```
